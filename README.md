# Web_App
Aplicação web: Front-End e Back-End
O projeto consiste em um Web App no qual faz a autenticação de usuários através do login. Além disso, o sistema também faz a contagem dos acessos, bloqueio do usuário caso erre a senha, manutenção de senha e cadastro de novos usuários.
